    Komponen Arsitektur Komputer INTERNAL

    PROSESOR

    merupakan otak dari komputer. Prosesor bertanggung jawab untuk menjalankan instruksi-instruksi yang diberikan oleh program, serta mengolah data yang diperlukan. Prosesor terdiri dari beberapa bagian, seperti unit kontrol, unit aritmatika dan logika (ALU), register, cache, dan bus.

    MEMORY

    merupakan tempat penyimpanan sementara untuk program dan data. Memori terdiri dari beberapa jenis, seperti Random Access Memory (RAM), Read Only Memory (ROM), dan memori sekunder. Memori berfungsi untuk menyimpan instruksi-instruksi dan data yang sedang diproses oleh prosesor, serta menyimpan informasi-informasi penting yang dibutuhkan oleh sistem komputer.

    Input/Output (I/O)

    merupakan perangkat yang digunakan untuk berkomunikasi dengan dunia luar. I/O terdiri dari beberapa jenis, seperti keyboard, mouse, monitor, printer, scanner, speaker, dan lain-lain. I/O berfungsi untuk menerima masukan dari pengguna atau sumber lain, serta memberikan keluaran kepada pengguna atau tujuan lain.

    KOMPONEN ARSITEKTUR KOMPUTER EKSTERNAL

    Komponen eksternal arsitektur komputer, yaitu perangkat keras, perangkat lunak, dan jaringan. Perangkat keras adalah perangkat fisik yang mendukung fungsi komputer, perangkat lunak adalah kumpulan program yang mengendalikan dan mengatur komputer, dan jaringan adalah kumpulan komputer yang terhubung melalui media transmisi.

    Central Processing Unit

    JENIS CPU

    CPU dapat diklasifikasikan berdasarkan beberapa kriteria, seperti arsitektur,kecepatan, jumlah bit, jumlah core, dan merek. Berikut adalah beberapa contoh jenis CPU berdasarkan kriteria tersebut

    REGISTER CPU

    Register CPU adalah komponen CPU yang berfungsi untuk menyimpan data sementara yang sedang diproses oleh CPU. Register memiliki kapasitas yang kecil namun kecepatan yang tinggi. Register dapat dibedakan menjadi beberapa jenis

    KOMPONEN UTAMA
